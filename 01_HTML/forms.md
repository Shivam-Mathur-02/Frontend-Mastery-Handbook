# HTML Forms

## What is a Form?

Forms collect user input.

Examples:

- Login
- Registration
- Contact form
- Search

---

## Common Elements

- form
- input
- textarea
- select
- button
- label

---

## Example

```html
<form>

<label for="name">

Name

</label>

<input
id="name"
type="text">

<label for="email">

Email

</label>

<input
id="email"
type="email">

<button type="submit">

Submit

</button>

</form>
```

---

## Common Input Types

- text
- email
- password
- number
- date
- checkbox
- radio
- file

---

## Best Practices

- Always use labels.
- Use appropriate input types.
- Use submit buttons.
- Validate user input.

---

## Common Mistakes

❌ Placeholder instead of label.

❌ Missing type attribute.

❌ Button without type.
