# HTML SEO Basics

## What is SEO?

SEO (Search Engine Optimization) helps search engines understand and rank your website.

---

## Important HTML Elements

### Title

```html
<title>Shivam Mathur | Frontend Developer</title>
```

---

### Meta Description

```html
<meta
name="description"
content="Frontend Developer Portfolio">
```

---

### Heading Structure

```
H1

↓

H2

↓

H3
```

Use only one H1.

---

### Images

Always write meaningful alt text.

---

### Semantic HTML

Search engines understand

- header
- nav
- main
- article
- footer

better than generic divs.

---

## Best Practices

- One H1
- Proper heading hierarchy
- Descriptive title
- Meta description
- Semantic HTML
- Alt text

---

## Common Mistakes

❌ Multiple H1

❌ Missing title

❌ Missing meta description

❌ Using only div elements
