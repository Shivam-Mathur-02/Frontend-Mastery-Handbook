# Grid Gap

## gap

Creates space between rows and columns.

```css
gap:20px;
```

---

## row-gap

Space only between rows.

```css
row-gap:30px;
```

---

## column-gap

Space only between columns.

```css
column-gap:20px;
```

---

## gap vs Margin

Gap

✔ Between Grid Items only

✔ Cleaner

✔ No outer spacing

Margin

✔ Belongs to individual elements

✔ Can create unwanted edge spacing

---

## Best Practice

Prefer using gap instead of margins for Grid layouts.
