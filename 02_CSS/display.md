# CSS Display

## Block

Examples

- div
- section
- article
- h1
- p

Properties

- Starts on a new line
- Takes available width
- Width works
- Height works

---

## Inline

Examples

- span
- a
- strong

Properties

- Flows with text
- Width ignored
- Height ignored

---

## Inline-block

Properties

- Flows inline
- Width works
- Height works

---

## When to Use

Block → Layout

Inline → Text

Inline-block → Small UI elements
